connman-gtk
===========

GTK GUI for ConnMan.

![screenshot-1](https://raw.githubusercontent.com/jgke/jgke.github.io/connman-gtk/connman-gtk-1.png)
![screenshot-2](https://raw.githubusercontent.com/jgke/jgke.github.io/connman-gtk/connman-gtk-2.png)

Dependencies
------------

 * intltool
 * GLib
 * GTK >= 3.14

Installation
------------

	./autogen.sh
	./configure
	make
	make install
